# CmdAlias Plugin

A Minecraft plugin that allows players to create custom command aliases.

## Build

```bash
mvn clean package
```

The compiled JAR file will be in the `target/` directory.

## Installation

1. Build the plugin using Maven
2. Copy the `cmdalias.jar` to your Minecraft server's `plugins/` directory
3. Restart the server

## Usage

```
/alias <alias> <command>
```

Example:
```
/alias warp teleport
```
